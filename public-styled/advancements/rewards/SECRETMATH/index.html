Quick Interpretation

These graphs show the whole Lucky Chest system from multiple angles: final level distribution, rarity distribution, level+rareness combinations, point expectation, badge completion value, specific badge timelines, and gem conversion value once badges are maxed.

Fixed Odds

The Lucky Chest has no player skill path. Final level odds, rarity odds, badge odds, and point EV are all fixed by code.

All Badges Unlocked

This dashboard ignores rank restrictions and treats every rarity as fully available at all times.

Specific Badge vs System

One chest creates decent total system progress, but one exact badge still takes much longer because progress is spread across many targets.

Gem Engine

Lucky Chests only become gem-producing once the chosen badge is already level 8.
